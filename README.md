# React MaterialUI Dashboard

## 使用スタック

- Node.js: v20.10.0
- React: v18.2
- MUI: v5.x
- Vite: v5.1.4

## 初回セットアップ

1. クローン

   ```bash
   git clone git@github.com:react-ts-practice-mk/react-mui-vite.git
   ```

2. 依存関係インストール

   ```bash
   npm install
   ```

3. 開発サーバー起動

   ```bash
   npm run dev
   ```
